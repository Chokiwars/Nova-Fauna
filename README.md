# ⚙️ Como executar o projeto

## 📥 Clone o repositório

```bash
git clone https://github.com/Chokiwars/Nova-Fauna.git
```

## 📂 Acesse a pasta

```bash
cd nova-fauna
```

## 📦 Instale as dependências

```bash
npm install
```

## ▶️ Execute o projeto

```bash
npm run dev
```

---

# 🛠 Scripts disponíveis
